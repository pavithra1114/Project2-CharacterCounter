# Project 2 - Real-Time Character Counter ✍️

This project is a simple web application that allows users to type a message in a textarea and displays a real-time character counter.

It shows how many characters are typed and how many are remaining out of 200 characters.

---

## 🚀 Features

- Textarea input box
- Maximum character limit: **200**
- Live character counter update while typing
- Displays typed characters like: **150/200**
- Prevents typing after reaching the limit
- Shows warning message when character limit is reached

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
